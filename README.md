# Customer Churn Prediction System

## 📌 Overview
This project predicts customer churn using a RandomForest classifier with hyperparameter tuning.

## 🚀 Features
- Data preprocessing pipeline
- Feature scaling and encoding
- RandomForest with GridSearchCV
- Model persistence using joblib
- Streamlit web application deployment

## 📊 Model Performance
- Accuracy: ~79%
- ROC-AUC: ~0.83

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
